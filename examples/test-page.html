🧪 Right Panel HTTP Preview Test Page

📋 测试说明

此页面用于测试 GoTTY Right Panel 的 HTTP 预览功能。

在 GoTTY 界面中：

  1. 点击右侧操作面板按钮（📋）
  2. 切换到 "HTTP" 标签页
  3. 输入此页面的 URL：http://localhost:8000/examples/test-page.html
  4. 点击 "Load" 按钮
  5. 页面应该在此 iframe 中显示
