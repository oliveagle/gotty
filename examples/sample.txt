========================================
GoTTY Right Panel - File Preview Test
========================================

此文件用于测试 GoTTY Right Panel 的文件预览功能。

使用方法:
---------
1. 在 GoTTY 界面中，点击右侧操作面板按钮（📋）
2. 切换到 "File" 标签页
3. 在文件路径输入框中输入此文件的绝对路径
4. 点击 "Load" 按钮
5. 文件内容应该在预览区域显示

测试路径示例:
--------------
/Users/oliveagle/ole/repos/github.com/oliveagle/gotty/examples/sample.txt

功能测试清单:
-------------
[ ] 文件内容正确显示
[ ] 支持中文文本
[ ] 支持特殊字符
[ ] 支持长文本滚动
[ ] 支持代码高亮（如果是代码文件）

支持的文件类型:
--------------
- .txt - 纯文本文件
- .md - Markdown 文件
- .json - JSON 文件（会自动格式化）
- .yaml / .yml - YAML 配置文件
- .js / .ts - JavaScript/TypeScript 代码
- .go - Go 语言代码
- .py - Python 代码
- .html / .css - Web 文件
- .log - 日志文件
- .xml - XML 文件
- .toml - TOML 配置文件
- 以及其他文本格式

限制:
-----
- 文件大小限制：1MB
- 不支持二进制文件
- 不支持需要特殊解码的文件
